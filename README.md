🚀 My Portfolio
Modern portfolio website built with Next.js 14, TypeScript, Tailwind CSS, and Radix UI.

✨ Features
    ⚡ Next.js 14 with App Router

    🎨 Modern glassmorphism design

    📱 Fully responsive

    🎯 TypeScript for type safety

    ⚡ Fast animations with Framer Motion

    🛡️ Accessible components with Radix UI

🚀 Quick Start
bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
